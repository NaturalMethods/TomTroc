# TomTroc
Projet 4 du parcours Développeur d'application full stack

## Pour utiliser ce projet :

- Commencer par cloner le projet.
- installez le projet chez vous, dans un dossier exécuté par un serveur local (type MAMP, WAMP, LAMP, etc...)
- Une fois installé chez vous, créez un base de données vide appelée : "TomTroc"
- Importez le fichier TomTroc.sql dans votre base de données qui est un export d'une base de donnée mariaDB.

## Lancez le projet ! 

Pour la configuration du projet, éditez le fichier config.php (dans le dossier config).
Ce fichier contient notamment les informations de connexion à la base de données (DB_HOST,DB_NAME...). 

Pour vous connecter avec un utilisateur sur le site, vous pouvez tout simplement en créer un depuis la page d'enregistrement sinon le mot de passe des utilisateurs est Azerty+35 par défaut.

Ce projet a été réalisé avec PHP 8.5. Bien que d'autres versions de PHP puissent fonctionner, il n'est pas garanti que le projet fonctionne avec des versions antérieures.
